XINANJIANG MODEL

Distributed Hydrologic Model
The Xin'anjiang model is a hydrological model proposed by Hohai University and a rare hydrological model with world influence in China. The Xin'anjiang model is a dispersive model that can be used for wet seasons in wet and sub-humid areas. When the basin area is small, the Xin'anjiang model uses a lumped model, and when the area is large, a block model is used. It divides the whole watershed into a number of block unit basins, and calculates the production and sinking flow for each unit basin, and derives the export flow process of the unit basin. Then carry out the river flood calculation below the exit and find the flow process of the river basin exit. By summing the outflow processes of each unit basin, the total outflow process of the basin is obtained.
